llbe(Lobs Line Based Editor) is a [line editor](https://en.wikipedia.org/wiki/Line_editor) inspired by [ed](https://en.wikipedia.org/wiki/Ed_(software)) 

## Compiling
Make with 
```sh
make install
```

## How to us 
run the command 
```sh
./llbe 'filename'
```
to start the editor \
Get commands using the h command
